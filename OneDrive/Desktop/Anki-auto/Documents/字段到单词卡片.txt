1. 需要什么样的数据结构作为输入

目前了解来看卡片本身貌似是一个字典，

note = {
    'deckName': 'MyDeck',                # 牌组名称
    'modelName': 'Basic',                # 模板名称
    'fields': {                          # 字段及其内容
        'Front': 'What is the capital of France?',
        'Back': 'Paris'
    },
    'tags': [],                          # 标签，可以为空
    'options': {'allowDuplicate': False} # 选项，如是否允许重复
}

结构大概长这个样子

2.自动化的基本原理是什么

anki的插件ankiconnect可以通过http请求把操作指令发给anki

3.需要的python包

request和json

原理：发送 POST 请求：使用 requests.post 方法可以向 AnkiConnect 服务器发送 POST 请求，附带请求负载（payload）。
处理响应：requests 可以轻松地处理来自服务器的响应，包括解析响应内容（通常是 JSON 格式）。

JSON：一种数据交换格式，用于在不同编程语言或系统之间传输数据。JSON 是一种文本格式，具有严格的语法规范。
Python 字典：是 Python 内建的数据结构，用于在 Python 程序内部存储和操作
